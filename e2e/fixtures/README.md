# Fixtures para Tests E2E

Esta carpeta contiene archivos de prueba para los tests E2E.

## Archivos necesarios:

### test-emo.pdf
Archivo PDF de prueba para tests de subida de EMO. Debe ser un PDF válido pero pequeño (< 1MB).

### large-file.pdf
Archivo PDF grande (> 10MB) para probar validación de tamaño máximo.

### test.exe
Archivo con extensión inválida para probar validación de tipo de archivo.

## Nota:
Los tests crearán estos archivos automáticamente si no existen, pero es recomendable tener archivos reales para pruebas más realistas.
